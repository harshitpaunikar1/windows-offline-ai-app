# Windows CPU-Only Offline AI Application

> **Domain:** Logistics

## Overview

Offline AI system for field teams on Windows tablets with local model inference, image preprocessing, OCR, and resilient background synchronization when connectivity returns.

- Python
- ONNX Runtime
